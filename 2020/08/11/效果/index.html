布局

上下左右top、bottom、left、right
设定一个值，套用到上下左右四个方向
设定两个值，套用到上下、左右
设定三个值，上、左右、下
设定四个值，上右下左
或直接指定，如padding-left
auto,自适应，absolute无效

padding,margin

padding:内，背景颜色，触发效果等均生效。
margin:外，不生效。
top、bottom、left、right:直接移动位置，影分身。

position

static

默认模式，不受top、bottom、left、right影响，接着文档流下去

absolute

不受同级其他元素的影响，存在于父类的固定位置，受滚动条影响。

relative

类似于static，受top、bottom、left、right影响，子类absolute遵循relative父类的规则，而不遵循static父类的规则。

fixed

类似于absolute，不受滚动条影响

sticky

类似于relative（待考），可设置边界，滚动至边界即不在滚动。

animation

值 描述
animation-name 规定需要绑定到选择器的 keyframe 名称
animation-duration 规定完成动画所花费的时间，以秒或毫秒计
animation-timing-function 规定动画的速度曲线
animation-delay 规定在动画开始之前的延迟
animation-iteration-count 规定动画应该播放的次数
animation-direction 规定是否应该轮流反向播放动画

##animation-fill-mode: forwards:结束时定格
